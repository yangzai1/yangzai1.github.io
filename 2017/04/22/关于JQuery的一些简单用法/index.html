jQuery的一些用法

jQuery的一些用法

（一）：获取可视区和物体的宽高

获取览器显示区域的高度：$(window).height();
获取浏览器显示区域的宽度：$(window).width();
获取页面的文档高度：$(document).height();
获取页面的文档宽度：$(document).width();
获取滚动条到顶部的垂直高度：$(document).scrollTop();
获取滚动条到左边的垂直宽度：$(document).scrollLeft();

（二）：jQuery的循环

1：For原生： for(var i = 0; i<10;i++){}
2：jQuery：$.each循环一些有意义的事情

A: 循环处理一组元素

$('li')       $('.on') 
[]-->表示为数组 {}———>表示为JSON
$.each(要循环的东西,function(index,element){  });
$.each({name1:v1,name2,v2},function(i,v){i  --  json中的  namev --  json中的 value});

B: 关于each循环，另外一种写法：

$('li').each(function(i,v){//每一个做处理});

（三）：字符串：

原生js: charAt() substring split indexOf lastIndexOf toLowerCase……

1:原生：trim()   --  去除空格，只去除首尾的空格
2:jQuery:  $.trim(str2)    --  去除空格，只去除首尾的空格

（四）jQuery的get请求

1：$.ajax(); 
2：$.get();  //发get请求  
3：$.post(); //发post请求 
4：$.getJSON();

（五）：jquery插件：http://jqueryui.com/（jQuery的一个插件地址）

 1：自己想写插件：
 A: $.fn.名字= function(){  }
 B: this  --  插件中的就是jquery对象 ，不再是一般情况下的原生对象

2:批量写插件：
  $.fn.extend({  toRed:function(){},..});

注意： jquery 1.4 版本以后，JSON中不允许使用单引号
注意： document.write以后，页面上就没东西了，不要再试图获取元素了
