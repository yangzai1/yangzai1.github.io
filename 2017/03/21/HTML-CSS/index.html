HTML/CSS

学习
  • CSS
  • HTML

一、标签分类：

       1.行内标签：
        span,a,i,em,b,strong,
        1)共处一行
        2)不支持宽高
        3)受空格和回车影响
    2.块元素
        p,div,h1-h6,ul,ol,dl,li,dt,dd
        1)独占一行
        2)支持宽高
        3)不受空格和回车影响
    3.行内块
        img,input
        1)共处一行
        2)支持宽高
        3)受空格和回车影响
元素转换：
    1)行内元素转成块元素    display:block;
    2)块元素转成行内元素    display:inline;(尽量不用)
    3)转行内块            display:inline-block;
书写规范：
    1)双标签必须闭合
    2)单标签合理闭合(/)
    3)合理嵌套和缩进
        a.块元素可以包块和行内(p只能包行内，不能包块)
        b.行内元素可以包行内(不要包自己)*(a可以包块,一般把a转化成块)*
    4)所有字母都小写

二、样式：

    1)行内        <div style=""></div>
    2)内联样式    <style></style>
    3)外联样式    <link href="" rel="stylesheet"
常见样式：width,height,border
border:(重点)
    1)合写：border:1px solid red;
    2)分写：border-width:粗细
            border-style:solid/dashed/dotted
            border-color:颜色
        border-top:上
        border-right:右
        border-bottom:下
        border-left:左
颜色表示法：
    1)单词    red/blue/pink/yellow/green/black/gray
    2)十六进制 a-f 0-9 
            #000000;  #000; (缩写)
    3)rgb(0-255,0-255,0-255)

三、背景样式

    background
        分写：background-color:
              background-image:url();
              background-repeat:no-repeat/repeat-x/repeat-y
              background-position:
                  1)百分比
                  2)像素 x轴 y轴
                  3)单词 left/right/center/top/bottom
                  4)混合 区分顺序
        合写：background:url() no-repeat 0 0 #ccc;
文字相关样式：
    font-size:字体大小
    font-family:字体类型  Microsoft Yahei
    font-weight:bold/normal;字体加粗/不加粗
    font-style:italic/normal;字体倾斜/不倾斜
    color:颜色
    合写：font: bold italic 30px/60px '宋体'；
文本水平对齐方式：
    text-align:center/left/right;
    行高：line-height:height;
    text-decoration:overline/line-through/underline/none;
em单位
    1)text-indent:2em;
    2)text-indent:-999; 隐藏logo上文字
    3)限制一行几个字符
文字不折行
    white-space:nowrap;    不折行
    overflow:hidden;    溢出隐藏
    text-overflow:ellipsis;    省略号

四、盒子模型

    宽：width+padding*2+border*2
    高: height+padding*2+border*2
padding:
    1个值    上下左右
    2个值    上下 左右
    3个值    上 左右 下
    4个值    上右下左
方向:
    padding-left/right/top/bottom;
margin(同上)
margin负值-----热门单击游戏
关于margin：
    1)拖拽
        解决：
            给父级加overflow:hidden;
            给父级加border;
            给父级加padding；
    2)塌陷    取得是margin-bottom或margin-top的最大值

五、清除浏览器默认样式

*{padding:0; margin:0; font-size:12px;}
ul,ol,li,dl,dt,dd{ padding:0; margin:0;}
li{ list-style:none;}
h1,h2,h3,h4,h5,h6{ font-weight:normal;}
a{ text-decoration:none; color:#000;}
i,em{ font-style:normal;}
b,strong{ font-weight:normal;}
.clearfix:after{ display:block; content:''; clear:both;}
.clearfix{zoom:1;}
.fl{ float:left;}
.fr{ float:right;}
img{display:block;}或者img{vertical-align:top;}
a img{ border:none;}
input,textarea{outline:none;}
table{ border-collapse:collapse;}
th,td{ border:1px solid #ccc; padding:0;}

    a:link{}  默认

    a:hover{} 鼠标悬停
    a:active{} 按下

八、浮动:

float:left/right;
    作用：
    放在一行
特点：
    1)脱离文档流(半脱离)
    2)改变元素类型，使其支持宽高
    3)顶对齐
    4)文本环绕
三要素(重点)
    1)给父级清浮动
    2)同级浮动，其他也要浮动
    3)浮动元素尽量设置宽高
清除浮动：
    1)overflow:hidden;
    2)在浮动元素后边加一个空的块元素
        <div style="clear:both"></div>
    3).clearfix:after{display:block; content:''; clear:both;}
       .clearfix{zoom:1;}

九、定位：

1.绝对定位
    position:absolute; left:0; right:0;
    特点：
        1)完全脱离文档流
        2)改变元素类型支持宽高
    参照物：
        1)当父级没有定位的时候，参照物是body
        2)当父级有定位的时候，参照物是父级
2.相对定位
    position:relative;
    特点：
        1)不脱离文档流
        2)不改变元素类型
    参照物：
        本身原来的位置---自己
3.固定定位
    position:fixed;
    特点:
        1)完全脱离文档流
        2)改变元素类型，支持宽高
    参照物：
        屏幕可视区
层级：
    z-index:999;
    普通元素<浮动元素<定位元素<定位+z-index;
    *z-index 只能应用在定位元素上
    *父级的层级永远不会发生变化
让一个已知宽高的div垂直水平居中
    div{
        width:300px;
        height:300px;
        position:absolute;
        left:50%;
        top:50%;
        margin-left:-150px;
        margin-top:-150px;
    }
让一个图片在块里面垂直水平居中
    <div>
        <img src="" />
        <span></span>
    </div>
    div{
        width:300px;
        height:500px;
        border:1px solid red;
        text-align:center;
    }
    img{
        vertical-align:middle;
    }
    span{
        display:inline-block;
        vertical-align:middle;
        height:100%;
    }

十、透明层

opacity:0.3;                0-1;
filter:alpha(opacity:30);    0-100
*必须写兼容 filter:---
*透明层非常强悍，可以把里面的东西全部变透明

十一、常见兼容

 1.行高
     解决：line-height:100%;或者line-height:字体大小；
 2.双边距
     浮动的方向和margin的方向一致时
     解决：
         给浮动元素加display:inline;
 3.img 3像素bug
     解决：
         display:block;/vertical-align:top;
 4.透明层
     filter:alpha(opacity:40);
 5.a img IE 下出现蓝色边框
     解决：
         a img{border:0;}
6.margin负值 改变元素位置多余的在IE下不显示
 解决：给它加 position:relative;

十二、表单

    <form action="地址" method="get/post">
        <input type="text" value="" name="" />
        <input type="password" name="" value="" />
        <input type="radio" name="" value="" id="" />
        <label for=""></label>
        <input type="checkbox" name="" value="" />
        <select name="">
            <option value=""></option>
        </select>
        <input type="file" />
        <input type="button" />
        <input type="submit" />
        <input type="reset" />
        <textarea></textarea>
    </form>
<form action="模拟百度.html" method="get"></form>
action    数据提交的地址
method    提交的方式
get    地址栏    不安全
post    后台    相对安全一点
<input type="password" name="num"/>     （密文输入框）密码框
<input type="text" name="num"/>   文本框
name---------想提交必须给它一个名字
<input type="radio" name="sex" id="id"/>单选框
*（用相同的name才可以单选）
<label for="nv">女</label>
for="ID"文字可以选择
<select>下拉框
    <option>下拉框内容</option>
</select>
input         输入框                 <input type="" />
type         类型
radio         单选                 <input type="radio" />
checkbox     多选                 <input type="checkbox" />
text         文本                 <input type="text" />
    textarea                   文本域                                                       <textarea></textarea>
password     密码                 <input type="password" />
submit         提交                 <input type="submit" />
button         按钮                 <input type="button" />
reset         重置                 <input type="reset" />
vertical-align:    垂直对齐方式
middle;        居中
top；        上边
bottom        下边            
outline:none;    取消焦点(取消表单边框)
resize:none;    取消文本域拖拽
    *清除默认样式
        input,textarea{outline:none;}
        textarea{resize:none;}
    *单选、多选、下拉框不要设置样式

    <frame />
