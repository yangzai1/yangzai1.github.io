1.css怎么写 js就怎么写

但是有个东西例外:
class className(不加style)
margin-left marginLeft(复合样式首字母大写)
单引号里面写双引号
双引号里面写单引号

2.函数声明和调用

声明:function 函数名() {
        //code
    }
    调用:函数名()

3.事件: 任何标签都能加事件 操作谁给谁加事件

点击事件:onclick
鼠标移入:onmouseover
鼠标移出:onmouseout
鼠标按下:onmousedown
鼠标抬起:onmouseup
select下选中的

4.document.body能直接选中body

**5.if判断–流程控制语句
if（条件）{语句1}； 简写：条件&&语句1
if（条件）{语句1}else if（条件）{语句2}; 简写：条件？语句1：语句2；（三目运算符）

switch（条件）{
case 情况1：
语句
break；
case 情况2：
语句
break;
default：
语句
break；
}

**break； 中断本次执行
for(var i=0;i<10;i++){ if(i==3){ break; //中断本次循环}
alert(i); //0,1,2
}

**continue； 跳过本次执行
for(var i=0;i<10;i++){ if(i==3){ continue; //跳出本次循环}
alert(i);//0,1,2,4,5,6,7,8,9
}
= 赋值 == 判断比较 === 全等

6.style 玩的都是行间

**7.传参:
(一个参数) 声明:function 名字(参数){
//code
//参数
}
调用:名字(参数)

(多个参数) 声明:function 名字(参数1,参数2){
//code
//参数1,参数2
}
调用:名字(参数1,参数2)

8.标示符和表达式:

.  标示符  . 只能操作已有属性
[] 表达式     只要 . 能干的  [] 都能干

9.获取元素

获取一组元素:
document.getElementsByTagName(‘标签’)
获取id
document.getElementById(‘id’)
通过class选取一组元素
document.getElementsByClassName(‘class’)

10.数组(Array):

var arr = [1,2,3,4,5]
length:数组的个数   从1开始
下标: 从0开始    arr[0]

解决方法:var _this=this;
