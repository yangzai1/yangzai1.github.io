91.事件绑定

作用：给同一个元素加同一个事件加多次。
    obj.addEventListener(sEv,fn,false);
    兼容：IE9+，chrome，firefox
    obj.attachEvent('on'+sEv,fn);
    兼容：IE10-
事件绑定兼容写法
if(obj.addEventListener){
    obj.addEventListener(sEv,fn,false);
}else{
    obj.attachEvent('on'+sEv,fn);
}

92.解除事件绑定

obj.removeEventListener(sEv,fn,false);
    兼容：IE9+，chrome，firefox
obj.detachEvent('on'+sEv,fn);
    兼容：IE10-
解除事件绑定兼容写法
if(obj.removeEventListener){
    obj.removeEventListener(sEv,fn,false);
}else{
    obj.detachEvent('on'+sEv,fn);
}
解除事件绑定，解除不了匿名函数
因为两个函数不想等
因为函数的真正写法是
new Function();

**往后只要加事件就加上事件绑定
