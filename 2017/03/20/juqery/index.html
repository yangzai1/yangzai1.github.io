juqery

学习
  • juqery

版本： 3.1.1

主版本号.次版本号.小版本号
小版本号：有代码改动  ，比如， 注释有改动，一点都没有影响
次版本号：BUG 的修复，优化，补充功能
主版本号：大改动，很有可能和原来完全不一样

官网：http://jquery.com/

1.x 兼容性：亚非拉
2.x 兼容性：欧美
3.x 新版
js实现的东西，jquery不一定全部能实现
jquery能实现的东西，用js一定能实现

$: – jquery 核心

写任何jquery 相关的东西，都是要用到 $

获取元素：

document.getElementById //  ByTagName
    --$('input')
    就是css 中的写法
    jquery里获取元素： css 中怎么写  jquery 就怎么写

原：  onclick
jq:    click  --  事件不带on

选择器

根据标签名：    $('input')
根据ID：         $('#btn1')
根据class:     $('.classname')

根据属性来获取：  [属性名='属性值']  //  '  "   交叉用：  外单引里双引，外双引里单引

.eq(2)   按索引找元素

伪类：

:first    获取第一个元素
:last    获取最后个元素
:even    匹配所有索引值为偶数的元素，从 0 开始计数
:odd    匹配所有索引值为奇数的元素，从 0 开始计数
:eq(索引值)----和下标类似
:lt(4)       小于索引值
:gt(4)    大于索引值
:contains(文本)  包含指定文本的 (如：包含 a 字母) 
:has(标签名) 谁包谁包含指定标签的

设置样式：

.css(‘样式名’); 获取
.css(‘background’,’yellow’); //设置一个
.css({‘background’:’yellow’,’width’:200}); 设置多个样式

显示： .show()

隐藏： .hide();

切换click事件，可以2个或多个： .toggle

mouseover mouseout–和原生的 on 事件，有同样的问题

移上，移走 切换的时候 ： .hover(用箭头函数会有问题)

淡入淡出： fadeIn fadeOut

下拉显示： slideUp slideDown

--不推荐使用

动画：animate –推荐使用–使用前先停止动画.stop（）

animate(params,[speed],[easing],[fn])
params:一组包含作为动画属性和终值的样式属性和及其值的集合
speed:三种预定速度之一的字符串(“slow”,”normal”, or “fast”)或
表示动画时长的毫秒数值(如：1000)
easing:要使用的擦除效果的名称(需要插件支持).默认jQuery提供”linear” 和 “swing”
fn:在动画完成时执行的函数，每个元素执行一次。
Params的属性值必须是数字 不是数字没办法运动
改变颜色:https://bitstorm.org/jquery/color-animation/jquery.animate-colors.js

样式：

.css('width');  //获取一个样式，带单位  200px
.css('background','yellow'); //设置一个
.css({'background':'yellow','width':200}); 设置多个样式

属性：

.attr('属性名')  //获取一个属性
.attr('name','value'); //设置一个属性
.attr({
    name1:'v1',
    name2:20
})    //批量设置属性

class操作：

addClass('class名')        添加class名
removeClass('class名')    删除class名

内容操作：

非表单元素的内容：

原生： .value

jq:

    .html()   获取innerHTML
    .html(设置的内容)  设置
表单标签：

原生： innerHTML

jq: .val() 获取内容

.val(设置的内容)  设置

原生对象 jquery对象

  • 原生对象的属性和方法，不能直接用于 jquery对象
  • jquery对象的属性和方法， 不能 直接用于 原生对象

原生 –> jquery $(原生)

jquery –> 原生 $(‘div’)[0]

this:

一般情况下， this  还是原生 js  中的this

索引：.index();

链式操作：

兄弟节点：siblings

位置：

原生： offsetLeft  offsetTop
        left   top

jquery:
    var l = $('#div1').position().left;
            var t = $('#div1').position().top;

位置：

原生：
    到定位父级：
        oDiv.offsetLeft
        oDiv.offsetTop
    绝对位置：
        getPos(oDiv).left
        getPos(oDiv).top
jquery:
    到定位父级：
        $(this).position().left 
        $(this).position().top
    绝对位置：
        $(this).offset().left
        $(this).offset().top

大小：

原生：
    盒模型：
        offsetWidth
        offsetHeight
jquery:
    普通宽：$(this).width()  //普通,不包含border和 padding
    普通高：$(this).height()

    $(this).innerWidth() //包含 普通+ padding
    $(this).innerHeight() 

    $(this).outerWidth() //包含普通+ padding+ border
    $(this).outerHeight()

绝对位置：

.offset().left； 距离可视区左边的宽度
.offset().top;

宽度/高度：

.width() 普通宽度
.innerWidth() 普通宽度 + padding
.outerWidth() 普通宽度 + padding + border

获取览器显示区域的高度 ： $(window).height();
获取浏览器显示区域的宽度 ：$(window).width();

获取页面的文档高度(html) ：$(document).height();
获取页面的文档宽度(html) ：$(document).width();

获取滚动条到顶部的垂直高度 ：$(document).scrollTop();
获取滚动条到左边的垂直宽度 ：$(document).scrollLeft();

DOM操作：

创建元素： $('<p>fffff</p>')

append:
    A.append(B);     // 在A里面追加B (后面)
appendTo:
    A.appendTo(B);   //把A放到B里面（后面）
prepend:
    A.prepend(B);  // 在A里面追加B （前面）
prependTo:
    A.prependTo(B);  //把A放到B里面（前面）
after:
    A.after(B);        //把B放到A后面 （外面）
before:
    A.before(B);       //把B放到A前面 （外面）
insertAfter:
    A.insertAfter(B);   //把A放到B后面 （外面）
insertBefore:
    A.insertBefore(B);  //把A放到B前面（外面）

.remove() 删除
.remove(表达式) 把匹配的元素删除

事件：

在jquery里，所有的事件，都是绑定的。

绑定，解绑，委托：

###1.7版本 开始，下面的方式，都弃用
bind() unbind() 直接绑定
live() die() 通过冒泡

delegate undelegate 更精确的小范围使用事件委托

on 加绑定

简单绑定： $(obj).on(‘事件名’,function(){});
传递数据： $(obj).on(‘事件名’,{数据},function(ev){
ev.data // 就是传过来的数据
});

多个事件： $(obj).on(‘事件名1 事件名2…’,function(){});

***多个事件用空格分开

on 加委托

off 解绑定

one 一次性事件

右键：contextmenu

阻止默认行为：

return false; -- 阻止默认行为 和 事件冒泡
ev.preventDefault();  --  阻止默认行为
ev.stopPropagation()  -- 阻止事件冒泡

*jquery里的事件详情 ev ，做了兼容处理，不用 ev||event

循环：

原生：  for(var i = 0; i<10;i++){}
jquery:  没有

循环一些有意义的事情

循环处理一组元素
    $('li')   $('.on') 
    []  数组
    {}   JSON
    $.each(要循环的东西,function(index,element){
    //index --  i
    //element  --  aLi[i]
      element  原生对象
    $(element) jq 对象
    });    index循环对应的数  


    $.each({name1:v1,name2,v2},function(i,v){
i  --  json中的  name
v --  json中的  value
    });

    关于each循环，另外一种写法：
$('li').each(function(i,v){
    //每一个做处理
});

###小工具：
$.browser.msie 是否是ie false/true
$.trim(str) 去除首尾空格
$.type 测试类型

jQuery: $.trim(str2) – 去除空格，只去除首尾的空格

ajax:
$.ajax();

$.get();  //发get请求
$.post();  //发post请求

$.getJSON();

jquery插件：
http://jqueryui.com/

自己想写插件：

自定义插件：$.fn.toRed = function(){
    //实现插件的代码
    在插件里，this  就 jquery对象。
}

this – 插件中的this 就是jquery对象 ，不再是一般情况下的原生对象

###批量写插件：

$.fn.extend({
    toRed:function(){
        this.css('','')
    },
    toBlue:function(){
        this.css('','')
    }
});

//可以单独放在js文件里面

*注意自定义插件放在window.onload($()) 外面

  • jquery 1.4 版本以后，JSON中不允许使用单引号
  • document.write 以后，页面上就没东西了，不要再试图获取元素
